How to use?

  1. Adjust Session and Break times as needed.
  2. Click 'New Pomodoro' to start the timer with those values.
  3. Click on the running time to pause/resume.

Note: Session/Break time can be changed at any time while timer is working. In that case, the new value will be initialized on the next timer rotation.
